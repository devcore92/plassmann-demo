# plassmann-demo

Unverbindlicher, mobiloptimierter **Website-Entwurf** für den Malerbetrieb
**Udo Plaßmann – Maler- & Lackierermeister** (Gütersloh-Isselhorst).
Mehrseitig, als wiederverwendbares Premium-Template angelegt.

> **Hinweis:** Unverbindlicher Design-Vorschlag, erstellt von Vincent Lokotsch.
> Keine offizielle Website der Firma; keine geschäftliche Verbindung. Inhalte
> öffentlich auffindbaren Quellen entnommen, nur zur Veranschaulichung.

## Seiten
- `index.html` – Startseite (Hero, Leistungen, Über uns, Prozess, Kontakt)
- `leistungen.html` · `ueber-uns.html` · `kontakt.html`
- `impressum.html` · `datenschutz.html` – Struktur als Platzhalter (ohne Rechtstext)
- `assets/styles.css` – gemeinsames Design-System · `assets/nav.js` – mobiles Menü

## Anpassen
- **Fotos:** Elemente mit der Klasse `frame` / `ph-label` sind Platzhalter –
  echtes Bild einsetzen (siehe `<!-- FOTO-SLOT -->`-Kommentare).
- **Farben/Schriften:** zentral in `:root` in `assets/styles.css`.
